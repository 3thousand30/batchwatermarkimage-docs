# BatchWatermark Image — Documentation

Official documentation for [BatchWatermark Image](https://3thousand30.com/batchwatermark-image.html), a batch watermarking tool for Windows by [3thousand30](https://3thousand30.com).

## What is BatchWatermark Image?

BatchWatermark Image adds text and logo watermarks to entire image folders in one run. Preview the result, adjust placement and opacity, then process the whole batch. Fully local — no cloud, no account, no subscription required.

## Key features

- **Text and logo watermarks** — apply either or both to your images
- **Batch processing** — watermark full folders, not one file at a time
- **Live preview** — see the result before processing the batch
- **Placement and opacity control** — fine-tune position, size, and transparency
- **Fully local** — no cloud upload, no account, no telemetry
- **No subscription** — buy once, use forever
- **Built for photographers, Etsy sellers, and content creators** protecting their work
