# BatchWatermark Image

**BatchWatermark Image** is a Windows desktop app by [3thousand30](https://3thousand30.com) that stamps image and text watermarks onto batches of photos — logos, copyright notices, signatures, and more.

Point it at a folder. Configure your watermark. Click Watermark. Done.

---

## How it works

1. **Select your input folder** — pick any folder of JPG, PNG, WebP, or TIFF images
2. **Configure your watermark** — overlay a logo image, stamp custom text, or use both at once
3. **Tune with preview** — compare original and watermarked versions side-by-side before processing
4. **Watermark** — the app composites the watermark onto each image and saves results to a subfolder

Image processing happens locally on your machine. The app may request the Inter UI font from Google Fonts for display consistency when an internet connection is available.

---

## Key features

- **Image watermark** — overlay a logo or PNG file at adjustable opacity and scale
- **Text watermark** — stamp any text using any installed system font, with color and size control
- **Both at once** — combine image and text watermarks in a single pass
- **9-position grid** — place your watermark in any corner, edge, or center
- **Repeat / Tile mode** — tile the watermark across the entire image
- **Margin and rotation** — fine-tune offset and angle (-180° to 180°)
- **Live preview** — watermarked vs original side-by-side before batch processing
- **EXIF metadata** — embed Copyright and Author fields into output files
- **Format conversion** — output as JPG, PNG, WebP, or keep the original format
- **Filename suffix** — optionally append a tag like `_wm` to output filenames
- **Skip existing files** — re-run safely without reprocessing already-watermarked images
- **Batch progress** — real-time counter with per-error logging and cancel support
- **Local image processing** — all processing happens on your device using Sharp
